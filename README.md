# سامانه انبارداری اداری (نسخه جایگزین Access)

این مخزن برای طراحی و پیاده‌سازی یک سامانه انبارداری مدرن با **C# / .NET** ایجاد شده است تا جایگزین فایل Access فعلی شود.

## هدف پروژه
- UX ساده برای کاربران غیر فنی (از جمله کاربر میانسال با تجربه محدود نرم‌افزاری)
- ظاهر مدرن و خوانا با تمرکز بر سرعت انجام کارهای روزمره
- داشبورد مدیریتی با نمودارها و اینفوگرافی داینامیک
- ثبت کامل گردش اموال: دریافت، خروج، انتقال، وضعیت، گزارش
- قابلیت توسعه‌پذیری و نگهداری بهتر نسبت به Access

## خروجی این فاز
در این فاز، یک نقشه راه اجرایی و طراحی سطح بالا آماده شده است:
- تعریف فازهای پیاده‌سازی
- پیشنهاد معماری فنی روی .NET
- مدل داده اولیه برای انبارداری اداری
- برنامه UX برای کاهش خطای کاربر

جزئیات در فایل زیر آمده است:
- [`docs/implementation-roadmap-fa.md`](docs/implementation-roadmap-fa.md)

## پیشنهاد تکنولوژی
- **Backend:** ASP.NET Core 8 Web API
- **Frontend:** Blazor Web App (Server) یا ASP.NET Core MVC (با توجه به سطح کاربری)
- **Database:** SQL Server
- **ORM:** Entity Framework Core
- **Auth:** ASP.NET Core Identity + Role-based Access
- **Charts:** ApexCharts یا Chart.js
